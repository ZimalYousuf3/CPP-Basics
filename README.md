## How to Use

### Compiling a Program
```bash
# Basic compilation
g++ program-name.cpp -o output

# With C++11 standard
g++ -std=c++11 program-name.cpp -o output

# With warnings enabled (recommended for learning)
g++ -Wall program-name.cpp -o output
```

### Running a Program
```bash
# Linux/Mac
./output

# Windows
output.exe
```
